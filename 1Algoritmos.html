Exploremos los conceptos básicos sobre algoritmos, variables y constantes.

Algoritmos

Un algoritmo es una serie de pasos ordenados que permiten resolver un problema o realizar una tarea específica.

  • Finitud: Un número definido de pasos.
  • Definición: Pasos claros y precisos.
  • Eficiencia: Consume pocos recursos.

1. Leer número 1
2. Leer número 2
3. Sumar ambos números
4. Mostrar el resultado

Variables

Las variables son contenedores que almacenan valores que pueden cambiar durante la ejecución de un programa.

  • Numéricas: Almacenan valores numéricos (ej. let edad = 25;).
  • Cadenas de texto: Almacenan texto (ej. let nombre = "Ana";).
  • Booleanas: Almacenan valores true o false.

Constantes

Las constantes son valores que no cambian durante la ejecución de un programa.

  • Ejemplos: Números como PI (3.1416) o valores booleanos (true o false).

const PI = 3.1416;
const esProgramador = true;
